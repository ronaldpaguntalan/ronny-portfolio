R2W Service Health Check

Ready2Work will offer enhanced tools for connectivity and service health assessment, empowering users to monitor ISP and connection speed, track 8x8 service status, assess Microsoft services status, and conduct VPN reachability tests seamlessly. Also, self-service logoff feature for Azure Virtual Desktop.

Blog Site - Frontend

A dynamic web platform built using Angular 16, Firebase, npm, TypeScript, and Bootstrap, designed for bloggers to create, publish, and share their content. It provides a user-friendly interface for readers to explore articles while offering bloggers a convenient space to craft and manage their posts.

Blog Site - Backend

The control center of your blog, powered by Angular 16, Firebase, npm, TypeScript, and Bootstrap. The Blog Dashboard empowers bloggers with tools to create, edit, and organize their content effortlessly. It also provides insights into article performance and user engagement, allowing bloggers to fine-tune their online presence effectively.

Angular Kanban Board

The Angular Kanban Board is a dynamic web application created using Angular, TypeScript, Angular Material, and Bootstrap. It offers an intuitive interface for managing tasks, complete with drag-and-drop functionality, and customization options. Ideal for efficient project and task management.

Captsone Project: HMIS

The system was developed using Microsoft Visual Basic 2012 as a programming language and MySQL 8.0 for the database, it also utilizes agile methodology. In terms of user access levels, the system has different modules wherein the components will collaborate exclusively through normalized interfaces.
